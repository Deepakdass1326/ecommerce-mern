# 🛒 Full-Stack E-Commerce App (MERN) – Work in Progress

This is a scalable e-commerce application built using the **MERN stack** (MongoDB, Express.js, React.js, Node.js), with JWT-based authentication, Redux Toolkit for state management, and PayPal payment integration (coming soon).

> 🚧 **Note:** This project is under active development.

---

## 🔧 Tech Stack

- **Frontend:** React.js, Tailwind CSS
- **Backend:** Node.js, Express.js (in `Server/`)
- **Database:** MongoDB with Mongoose
- **State Management:** Redux Toolkit
- **Authentication:** JWT
- **Payments:** PayPal (planned)
